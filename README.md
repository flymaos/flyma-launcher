# PandaOS Launcher

This is pandaos application launcher, comprising a full-screen and keypad operation, to meet the needs of different users.

## Dependencies

`sudo pacman -S gcc extra-cmake-modules qt5-base qt5-tools qt5-svg qt5-x11extras kwindowsystem`

## Build

```
mkdir build
cd build
cmake ..
make
sudo make install
```
